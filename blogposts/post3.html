Combinatorics in disguise part 2

PROBLEM: For a positive integer $n$ , denote by $ S(n)$ the number of choices of the sign $ +$ or $ -$ such that $ \pm 1 \pm 2 \pm... \pm n = 0$. Show that

$S(n) = \frac{2^{n-1}}{\pi} \int\limits_0^{2\pi} \cos(t) \cos(2t) ... \cos(nt)\, dt$

SOLUTION: The problem is quite a bit peculiar because it connects a counting thing with a very odd-looking integral. Here is the solution which is almost a sequel of this post.

First we use a generating function technique to compute $S(n)$.

Note that $S(n) = \text{coeff of }x^0 \text{ in } (x+\frac{1}{x})(x^2+\frac{1}{x^2})...(x^n+\frac{1}{x^n})$. Now for all $m\neq 0$ we can see if $x^a$ is present in the expansion of this expression then obviously $x^{-a}$ is present too. Now note that $2 \cos(kt) = e^{ikt}+e^{-ikt}$.

Now

$$\int\limits_0^{2\pi} \cos(t) \cos(2t) \dots \cos(nt)\, dt$$ $$= \int\limits_0^{2\pi} \prod\limits_{k=1}^n \left(\frac{e^{ikt}+e^{-ikt}}{2}\right) \, dt$$ $$= \frac{1}{2^n}\int \limits_0^{2\pi} \left[ \sum\limits_{\text{for suitable } m>0} (e^{imt}+e^{-imt})+S(n)\right]\,dt$$ $$= \frac{1}{2^n}\left(2\pi S(n) +\sum\limits_{\text{for suitable } m>0}\left[\int \limits_0^{2\pi} (e^{imt}+e^{-imt}) \,dt\right] \right)$$ $$= \frac{1}{2^n}\left(2\pi S(n) +\sum\limits_{\text{for suitable } m>0}\left[\left[\frac{e^{imt}}{im}-\frac{e^{-imt}}{im}\right]_0^{2\pi}\right] \right)$$ $$= \frac{1}{2^n}\left(2\pi S(n) +\sum\limits_{\text{for suitable } m>0}\left[\frac{(e^{2\pi i })^m -1 -(e^{-2\pi i })^m+1}{im}\right] \right)$$ $$= \frac{1}{2^n}\left(2\pi S(n) +\sum\limits_{\text{for suitable } m>0}(0) \right)$$ $$= \frac{1}{2^n}\left(2\pi S(n) \right)$$ $$= \frac{1}{2^{n-1}} \pi S(n)$$ $$\implies \boxed{S(n) = \frac{2^{n-1}}{\pi} \int\limits_0^{2\pi} \cos(t) \cos(2t) \dots \cos(nt)\, dt}$$

REMARK: Note that if $n = 4k+1 \text{ or } 4k+2$ then note that there are odd many odd numbers so the sum can never be $0$ as it always will be odd. Hence, we will have the integral $0$ for such $n$
